1.입력 양식:
입력 양식은 사용자에게 입력받는 공간을 의미합니다. 입력 양ㅅㄱ을 만들 떄 사용하는 태그입니다.
양식에 입력된 내용은 서버로 전송되어 구현된 웹 응용 프로그럄이 구동되어 데이터베이스와 연동
<fom 속성1=속성값 속성2=속성값...>
	<input type="~" name="전달할파라미터이름"
</form>

1_action: 입력 데이터의 전달 위치를 지정
2_method: 입력 데이터의 전달 방식을 선택
 (-get:url에 포함되어 전송(query String 형식)
 (-posty: 입력한 데이터가 눈에 보이지 않게 전송(url이 아니라 데이터 영역을 이용하여 데이터를 전송하기에 서버에 상관없이 
	피라미터 길이가 제한이 없다.)
3_button: 버튼을 생성
4_checkbox: 체크박스를 생성
5_file: 파일 입력 양식을 생성
6_hidden: 보이지 않게 합니다.
7_image: 이미지 형태를 생성
8_password: 비밀번호 입력 양식을 생성
9_radio: 라디오 버튼을 생성
10_reset: 초기화 버튼을 생성
11_submit: 제출 버튼을 생성
12_text: 글자 입력 양식을 생성  

2.공간분할 태그(레이아웃태그)-div(블록형식의 태그),span(라인형식의 태그)
	(1).div: block 형식으로 공간을 분할. 하나의 블록형식의 태그므로 개행포함 ex)p,ul안의 li
	(2).span: inline 형식으로 공간을 분할

3.div를 이용한 레이아웃
	<body>
		<div id="header">해더부분</div>
		<div id="menu">메뉴부분</div>
		<div id="content">정보내용부분</div>
		<div id="footer">꼬리말</div>
	</body>

4.시멘틱을 이용한 레이아웃
	-HTML5 시멘틱 구조 태그
	⊙header: 헤더
	⊙nav: 내비게이션
	⊙aside: 사이드에 위치하는 공간
	⊙section: 여러 중심 내용을 감싸는 공간
	⊙article: 글자가 많이 들어가는 부분
	⊙footer: 꼬리말
