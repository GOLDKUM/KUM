1. 선택자.
HTML 문서 중 특정 부분을 선택하여, 해당 부분의 속성을 변경하는 목적으로 사용
<html lang="en" xmlns="http://www.w3.org/1999/xhtml">
<head>
    <meta charset="utf-8" />
    <title></title>
    <style>
        div{background-color:#FFD800; border:5px solid #FF0000;}
        h1#id1{background-color:#2108ff}
    </style>
</head>
<body>
    <h1 id="id1">제목h1</h1>
    <p>본문 p</p>
    <div>
        <h1>div 밑의 h1 태그 부분</h1>
        <p>div 밑의 p태그 부분</p>
    </div>
</body>
</html>

2. tag선택자.
특정 태그를 선택하여 CSS속성을 적용할 수 있다.
지정된 태그명을 가지는 요소


3. id(#)와 class(.):
태그의 id와 class를 이용하여 CSS에 속성을 설정할 수 있다.
id은 '유일'한 요소, class는 '복수'의 요소

4.tag,id,class혼합 :
 tag,id,class를 혼합하여 CSS에서 속성을 설정 할 수있다.

5. 속성 선택자: tag의 속성을 선택하여 CSS의 속성을 설정할 수 있다.
(1) 선택자[속성] : 지정한 속성을 가지고 있는 요소를 찾아 스타일을 적용
(2) 선택자[속성=값]: 속성 값이 특정값과 같은 경우를 찾아 스타일을 적용
(3) 선택자[속성^=값]: 속성 값이 지정한 문자로 시작하는 속성값에 대해서만 스타일을 적용
(4) 선택자[속성$=값]: 속성 값이 지정한 문자로 끝나는 속성에 대해서만 스타일을 적용

6.후손선택자,자손선택자
자신의 한단계 상위에 속하는 요소를 부모 요소 
하위에 바로 아래단계 속하는 요소를 자손 요소
자신보다 하위에 속하는 요소를 후손요소

7.동위(형제)선택자
같은 부모를 가지는 요소들을 형제

8.반응선택자
마우스를 올려 놓았을 떄(hover상태)의 반응에 따라 속성
hover{/* li위에 마우스를 올렸을 떄 반응할 디자인(반응선택자)}

9.상태선택자
상태에 따라 CSS속성이 변화하는 설정을 할 수 있다,
ex.input태그의 경우 focus가 맞혀졌을 때
input:focus{/*상태선택자*/}
