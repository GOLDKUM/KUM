[ II ] HTML – I
☆태그란?
이 태그(기능)는 열린태그와 닫힌태그로 나눠진다. 그리고 태그에는 이름과 속성, 속성값이 있습니다.(예외 닫힌 태그가 필요없는 태그도 존재합니다)

ex)
	a href="http://www.daum.net">
	<!- 태그 이름 : A, 태그속성 :href, 태그값이 있다 -->

☆HTML 구조

(1)	<!DOCTYPE html>
	반드시 첫번째 줄에 있어야하며 HTML5문서임을 인식하게 만들어 줍니다.
(2)	<html  lang=”ko”>
	html태그의 시작을 알리고 lang 속성을 입력
(3)	<head>  …  </head>
	문서의 정보인 title이나 설정값들인 css, javaScript등의 환경설정에 관한 정보
(4)	<body>  …  </body>
	사용자에게 보이는 웹 문서 실제 정보
(5)	</html>

☆글자에 관련된 태그: 
H1(제목), p(본문), br(개행), a(앵커), b, I, sup, ins, del 태그

☆목록 태그 – ul,ol,li,dl,dt,dd 

☆표를 나타내는 태그 – table
tr(표 내부의 행 ),th(행 내부의 열. 제목 셀 ),td (행 내부의 열. 일반 셀 )
표형식의 레이아웃을 만드는 태그로, 예전에는 웹사이트의 전체적인 레이아웃을 만들었으나 현재는 회원가입폼이나 로그인폼, 게시판폼 사용

☆이미지를 출력하는 태그 – img
src : 이미지의 경로 지정
alt : 이미지가 없을 때 나오는 글자 지정
width : 이미지의 너비 지정 
height : 이미지의 높이 지정 

☆오디오 태그(HTML5에서만 추가) – audio
src : 오디오 파일의 경로 지정
preload : 오디오를 재생하기 전에 모두 불러올지 지정
autoplay : 오디오를 자동 재생할지 지정 (인터넷 익스플로어에서 작동)
loop : 오디오를 반복할지 지정
conrols : 오디오 재생 도구를 출력할지 지정

☆동영상을 출력하는 태그 - video
poster : 비디오 준비 중일 때의 이미지 파일 경로 지정
preload : 비디오를 재생하기 전에 모두 불러올지 지정
autoplay : 비디오를 자동 재생할지 지정
loop : 비디오를 반복 재생할지 지정
controls : 비디오 재생 도구를 출력할지 지정
width : 비디오의 너비를 지정
height : 비디오의 높이를 지정

☆detail과 summary  
detail : 세부정보나 추가 정보를 숨겨 놓았다가 펼쳐 보이는 기능
summary : 세부 항목의 제목또는 요약 내용
